Build a full-stack AI-powered weather prediction system.

Tech stack:

* Backend: Node.js with Express
* Frontend: HTML + CSS + Vanilla JavaScript (simple UI)
* Database: PostgreSQL
* AI logic: Basic rule-based system (structured for future ML upgrade)

Project goal:
Create a micro-climate weather system that:

* Fetches real-time weather data
* Stores historical weather data
* Uses simple AI logic to predict short-term weather
* Learns over time (structure ready for future ML integration)

========================
BACKEND REQUIREMENTS
====================

1. Create an Express server (server.js):

   * Runs on port 5000
   * Uses dotenv, cors, express.json()

2. Create routes:

   * GET "/" → returns "Weather AI running"
   * GET "/weather":

     * Accepts lat and lon
     * Fetches weather from Open-Meteo API
     * Passes data to AI service
     * Stores weather data in PostgreSQL
     * Returns:
       {
       weather: {...},
       prediction: "...",
       confidence: number
       }

3. Create modular structure:

   * /services/weatherService.js → fetch weather from API
   * /services/aiService.js → prediction logic
   * /db.js → PostgreSQL connection
   * /routes/weatherRoutes.js → route handling

4. AI Logic (aiService.js):

   * Use rule-based prediction:

     * If humidity > 80 and pressure < 1000 → "Rain likely soon"
     * If windspeed > 20 → "Storm possible"
     * Else → "Stable weather"
   * Return prediction + confidence score

5. Weather API:

   * Use Open-Meteo
   * Include temperature, windspeed, and weather data

========================
DATABASE REQUIREMENTS
=====================

1. Use PostgreSQL

2. Create table:

CREATE TABLE weather_data (
id SERIAL PRIMARY KEY,
latitude FLOAT,
longitude FLOAT,
temperature FLOAT,
windspeed FLOAT,
humidity FLOAT,
pressure FLOAT,
prediction TEXT,
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

3. Every time /weather is called:

   * Save data into this table

4. Structure code so historical data can later be used for ML training
